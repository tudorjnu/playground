Can gradient descent do the hard work for us?

This activity is the same as the previous one, except that this time you will not have to manually find suitable weights and biases, but you will be able to let the gradient-descent algorithm find good parameters for you!

  1. Click on the play button to start the neural network training.
  2. You will have to give the gradient-descent algorithm some non-zero initial values for the parameters!
    Use the popups as before to initialize weights and biases.

Explanation

In this activity, you use the gradient-descent algorithm to find good parameters for the neural network. It is quite impressive to see how quickly it reaches a near-perfect separation!

Gradient descent is a local optimization algorithm: The algorithm proceeds in rounds. In each round, it is computed how each weight and each bias influenced the final prediction of each data point. Based on that, and which data points are currently incorrectly classified, a suggested improvement (the “gradient”) is computed and added to the current value of all parameters. Each step makes the prediction slightly better (on the training data!).

Over time, the gradient-descent method reaches a state in which no further improvements happen (a stationary state); then the parameters remain unchanged.

An important limitation of such local optimization methods is that they depend on the starting state. For example, having all weights and biases equal to zero is a stationary state, but clearly not one that yields a good prediction! Luckily, for most neural networks it works just fine to start with randomly chosen initial values.

What do the colors mean?

Blue and gold are used throughout the visualization in slightly different ways, but in general blue shows negative values while gold shows positive values.

The data points (represented by small circles) are colored gold or blue, which correspond to positive one and negative one (the classes of data points).

In the hidden layers, the lines are colored by the weights of the connections between neurons. Gold shows a positive weight, which means the network is using that output of the neuron as given. A blue line shows that the network is assigning a negative weight.

In the output layer, the dots are colored gold or blue depending on their original values. The background color shows what the network is predicting for a particular area. The intensity of the color shows how confident that prediction is.
